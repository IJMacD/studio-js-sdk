int System ID of the room

Course

A series of lessons forms a course

Properties

id
int System ID of the course
title
string Title of the course
day
int 0-Sunday based day of the week this courses' lessons are usually on
startTime
string Usual start time of lessons in the format "0900"
endTime
string Usual end time of lessons in the format "1010"

Properties

id
int System ID of the lesson
course
Course Course this lesson belongs to

Date Start time and date of lesson
end
Date End time and date of lesson
tutor
Tutor Tutor who will teach this lesson
room

Properties

id
int System ID of the student
name
string Student's name as it appears in the system, sanitised
forename
string Given English name, if student has one
surname
string Family name of student
englishName
string Forename + surname
chineseName
string Three chinese names, without forename
