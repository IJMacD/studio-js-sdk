Defined in core.js:274

Useful function to set the name on any object with a name property.

Parses names to extract forename and surname as well as combinations for English and Chinese names. It recognises common Chinese surnames to avoid confusion with English name first vs. English name last.

This method modifies the original object by adding the properties: forename, surname, englishName and chineseName.

Parameters:

  • person Object

    Object with a name property
